sendSnap(blob, recipients, text, duration, cb)

Sends a snap to everyone in \c recipients with text \c text for \c duration seconds.

Parameters:
Name Type Description
blob SKBlob

The \c SKBlob object containing the image or video data to send. Can be created with any \c NSData object.
recipients Array.<string>

An array of username strings.
text string

The text to label the snap with. This text is not superimposed upon the image; you must do that yourself.
duration number

The length of the snap. It must be greater than \c 0 or an exception will be raised.
